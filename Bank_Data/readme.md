# Bank Marketing Prediction Using Decision Tree Classifier

## Project Overview

This project uses a Decision Tree Classifier to predict whether a bank customer will subscribe to a term deposit based on demographic and behavioral information.

The project is built using the Bank Marketing Dataset from the UCI Machine Learning Repository.

---

## Project Objective

To build a machine learning model capable of predicting customer subscription behavior and helping banks improve marketing campaign effectiveness.

---

## Dataset Information

- Dataset Name: Bank Marketing Dataset
- Source: UCI Machine Learning Repository
- Records: 45,211
- Features: 16 Input Features
- Target Variable: y

### Target Variable

| Value | Meaning |
|---------|---------|
| 0 | No Subscription |
| 1 | Subscription |

- Call duration strongly influences subscription decisions.
- Previous successful campaigns positively affect future subscriptions.
